# Video Belajar Server

## Cara mulai

1. install seluruh dependency

```
npm install
```

2. copy file dari `.env.example` ke `.env` lalu sesuaikan nilainya dengan milikmu

3. Jalankan perintah ini untuk memulai migrasi dan seed. Pastikan bahwa sebelumnya mysql server telah terinstall

```
npm run db:init
```

4. Jalankan untuk menjalankan server dalam mode development

```
npm run dev
```
